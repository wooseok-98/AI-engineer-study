# 💻 머신러닝(Machine Learning) VS 딥러닝 (Deep Learning)
---
## 머신러닝(Machine Learning)
- 데이터로부터 **패턴을 학습**하여 **예측, 의사결정** 수행
- 일반적으로 **사람이 직접 특징(feature) 설계**
- 예시:
    - Linear Regression
    - Logistic Regression
    - Decision Tree
    - Random Forest
    - SVM
    - KNN
---
## 딥러닝(Deep Learning)
- 여러 층으로 구성된 **인공신경망(Artificial Neural Network)** 을 기반으로 학습 진행
- 여러개의 **층(layer)** 을 사용하여 **특징(fearue) 자동 추출, 복잡한 패턴 학습**
- 이미지, 음성, 자연어 처리처럼 **비정형 데이터**에서 강력한 성능 
- 예시:
    - MLP
    - CNN
    - RNN
    - LSTM
    - Transformer

### 인공신경망 (Artificial Neural Network)
인간 **뇌의 신경 세포(뉴런)** 연결 구조를 모방한 수학적 모델
- **뉴런(Neuron) / 노드(Node):** 인공신경망의 기본 단위, 입력을 받아 처리하고, 출력을 생성
- **가중치(Weight):** 뉴런 간 연결 강도를 나타내는 값, 학습을 통해 조정
- **편향(Bias):** 뉴런의 활성화 여부를 조절하는 값
- **활성화 함수(Activation Function):** 뉴런의 출력을 결정하는 함수 (예: 시그모이드, ReLU, tanh)
- **층(Layer):** 여러 개의 뉴런이 모여 하나의 층을 구성
    - **입력층(Input Layer):** 외부로부터 데이터를 입력받는 층
    - **은닉층(Hidden Layer):** 입력층과 출력층 사이에 있는 층, 데이터의 추상적인 특징 학습
    - **출력층(Output Layer):** 최종 예측 결과를 출력하는 층
    
인공신경망의 **작동 방식**:
1. **입력:** 입력 데이터가 입력층의 뉴런에 전달
2. **전파:** 각 뉴런은 입력값에 가중치를 곱하고 편향을 더한 후, 활성화 함수를 적용하여 출력을 계산
3. **출력:** 출력층의 뉴런은 최종 예측값을 출력
4. **학습:** **역전파(Backpropagation)** 알고리즘을 사용하여, **예측값과 실제값의 차이(오차)** 를 줄이는 방향으로 **가중치 조정**
---
## 💡 요약
| 구분 | Machine Learning | Deep Learning |
|---|---|---|
| 관계 | 인공지능의 한 분야 | 머신러닝의 하위 분야 |
| 특징 추출 | 사람이 직접 feature engineering 수행 | 모델이 특징을 자동으로 학습 |
| 데이터 요구량 | 비교적 적은 데이터로도 가능 | 많은 데이터가 필요한 경우가 많음 |
| 계산량 | 비교적 적음 | 큼 |
| 모델 복잡도 | 상대적으로 단순 | 복잡 |
| 적용 분야 | 정형 데이터에 강함 | 이미지, 음성, 텍스트 등 비정형 데이터에 강함 |

# 📝 면접 예상 질문
<details>
  <summary>Q. 머신러닝과 딥러닝의 차이는 무엇인가요?</summary>

  딥러닝은 머신러닝의 하위 분야입니다.
  머신러닝은 보통 사람이 직접 feature engineering을 수행한 뒤 모델이 이를 학습하는 경우가 많습니다.  
  반면 딥러닝은 인공신경망을 사용하여 자동으로 특징 추출과 예측을 end-to-end로 학습합니다.  
  또한 딥러닝은 이미지, 음성, 텍스트 같은 비정형 데이터에서 강점을 보이지만, 일반적으로 더 많은 데이터와 계산 자원이 필요합니다.

</details>

<details>
  <summary>Q. 딥러닝은 왜 머신러닝의 하위 분야라고 하나요?</summary>

  딥러닝도 결국 데이터로부터 패턴을 학습하여 예측이나 분류를 수행하는 머신러닝의 한 방법이기 때문입니다.  
  즉, 머신러닝이라는 큰 범주 안에 여러 학습 방법이 있고, 그중 인공신경망을 깊게 쌓아 학습하는 방식이 딥러닝입니다.

</details>

</details>

<details>
  <summary>Q. 어떤 경우에 머신러닝을 사용하고, 어떤 경우에 딥러닝을 사용하나요?</summary>

  데이터가 많지 않고, 정형 데이터(tabular data)를 다루며, 해석 가능성이 중요할 때는 머신러닝을 우선 고려할 수 있습니다.  
  반대로 데이터가 많고 이미지, 음성, 텍스트처럼 복잡한 패턴을 자동으로 학습해야 할 때는 딥러닝이 적합합니다.  
  따라서 문제의 특성, 데이터의 형태, 연산 자원, 해석 가능성 등을 함께 고려해서 선택해야 합니다.

</details>

<details>
  <summary>Q. 머신러닝이 정형 데이터에 강한 이유는 무엇인가요?</summary>

  정형 데이터는 이미 사람이 이해할 수 있는 구조화된 feature로 이루어져 있는 경우가 많습니다.  
  그래서 선형 회귀, 로지스틱 회귀, 트리 기반 모델 같은 전통적인 머신러닝 알고리즘이 효과적으로 패턴을 학습할 수 있습니다.  
  또한 데이터 양이 많지 않아도 비교적 안정적으로 학습되는 경우가 많습니다.

</details>
