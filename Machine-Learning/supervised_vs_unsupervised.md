# 💻 학습(Supervised Learning) VS 비지도 학습(Unsupervised Learning)
---
## 지도 학습 (Supervised Learning)
- **정답(label)** 이 포함된 입력-출력 쌍 **(x, y)** 데이터를 사용하여 학습
- 입력 변수 **(x)** 와 출력 변수 **(y)** 간의 관계 학습 → 새로운 입력에 대한 출력 **예측**
- 유형:
    - **분류(Classification)**: 출력 변수가 **범주형** (예:생존/사망, 개/고양이)
        - **이진 분류(Binary Classification)**: 두 개의 범주로 분류 (예: 생존/사망)
        - **다중 클래스 분류(Multi-class Classification)**: 세 개 이상의 범주로 분류 (예: 붓꽃 품종 분류)
    - **회귀(Regression)**: 출력 변수가 **연속형** (예: 주택 가격, 주가, 온도)
- 예시:
    - Linear Regression
    - Logistic Regression
    - Decision Tree
    - Random Forest
    - SVM
    - Neural Network
    
---
## 비지도 학습 (Unsupervised Learning)
- **정답(Label)이 없는 데이터** 사용하여 모델 학습
- 데이터의 **구조나 패턴**을 찾거나, **유사한 그룹으로 묶는**데 사용
- 유형:
    - **군집화(Clustering)**: 유사한 데이터들을 묶어 **그룹(cluster)** 을 생성 (예: 고객 세분화)
    - **차원 축소(Dimensionality Reduction)**: 고차원 데이터를 **저차원으로 변환**하여 데이터를 요약하거나 시각화 (예: PCA)
    - **연관 규칙 학습(Association Rule Learning)**: 데이터 항목 간의 **연관성**을 찾는 기법입니다. (예: 장바구니 분석)
- Examples
    - K-Means
    - PCA
    - DBSCAN
    - Hierarchical Clustering
---
    
## 요약 (Summary)
| 구분 | 지도 학습 | 비지도 학습 |
|---|---|---|
| 데이터 | 정답(label) 있음 | 정답(label) 없음 |
| 목적 | 입력과 출력의 관계 학습 | 데이터의 구조나 패턴 발견 |
| 대표 작업 | 분류, 회귀 | 군집화, 차원 축소 |
| 예시 | 스팸 메일 분류, 주택 가격 예측 | 고객 세분화, 데이터 시각화 |
---

# 📝 예상 질문
## Interview Questions

<details>
  <summary>Q. 지도학습과 비지도학습의 차이는 무엇인가요?</summary>

  지도학습은 정답(label)이 포함된 데이터를 사용하여 입력과 출력의 관계를 학습하는 방식입니다.  
  반면 비지도학습은 정답이 없는 데이터에서 숨겨진 구조나 패턴을 찾는 방식입니다.  

</details>

<details>
  <summary>Q. 지도학습의 대표적인 예시는 무엇인가요?</summary>

  지도학습의 대표적인 예시로는 분류와 회귀가 있습니다.  
  분류는 출력값이 범주형인 문제로, 예를 들어 스팸 메일 분류나 개와 고양이 분류가 있습니다.  
  회귀는 출력값이 연속형인 문제로, 예를 들어 주택 가격 예측이나 온도 예측이 있습니다.

</details>

<details>
  <summary>Q. 비지도학습의 대표적인 예시는 무엇인가요?</summary>

  비지도학습의 대표적인 예시로는 군집화와 차원 축소가 있습니다.  
  군집화는 유사한 데이터들을 그룹으로 묶는 방식으로 고객 세분화에 활용될 수 있습니다.  
  차원 축소는 고차원 데이터를 저차원으로 줄여 시각화하거나 중요한 정보를 요약하는 데 사용되며, 대표적인 예로 PCA가 있습니다.

</details>

<details>
  <summary>Q. 지도학습과 비지도학습의 성능 평가는 어떻게 다른가요?</summary>

  지도학습은 정답이 있기 때문에 accuracy, precision, recall, RMSE 같은 명확한 평가 지표를 사용할 수 있습니다.  
  반면 비지도학습은 정답이 없으므로 군집의 응집도나 분리도 같은 내부 평가 지표를 사용하거나, 시각화와 도메인 해석을 통해 결과를 평가하는 경우가 많습니다.  
  즉, 지도학습은 정답 기반 평가가 가능하지만 비지도학습은 상대적으로 평가가 어렵습니다.

</details>

<details>
  <summary>Q. 강화학습은 지도학습, 비지도학습과 어떻게 다른가요?</summary>

  지도학습은 정답(label)을 통해 학습하고, 비지도학습은 정답 없이 데이터의 구조를 찾습니다.  
  반면 강화학습은 에이전트가 환경과 상호작용하면서 보상을 최대화하는 방향으로 행동을 학습합니다.  
  즉, 강화학습은 정답을 직접 주는 대신 보상 신호를 기반으로 최적의 행동 정책을 학습하는 방식입니다.

</details>
